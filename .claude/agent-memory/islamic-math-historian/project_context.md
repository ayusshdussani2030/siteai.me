The project is an IB MYP Year 3 Mathematics Exploration on Islamic Golden Age mathematics (8th–13th centuries).

A single-page static website was built at `/Users/ayusshdussani/code/math_website/index.html` on 2026-06-01. The site contains all of the following sections, with full academic content (no placeholders):

1. Hero — "The Islamic Golden Age of Mathematics"
2. Al-Khwārizmī and the origin of al-jabr (biography, etymology, why revolutionary, inheritance-law motivation)
3. Hindu-Arabic numeral conversion table for 1, 7, 19, 60, 144, 1000, 2026 — with thousands/hundreds/tens/units columns and the rule Value = Σᵢ dᵢ × 10ⁱ; also a Roman vs Greek vs Hindu-Arabic comparison table.
4. Side-by-side comparison of al-Khwārizmī's rhetorical/geometric solution to x² + 10x = 39 vs the modern completing-the-square and quadratic-formula solutions, with an inline SVG diagram of the completing-the-square construction.
5. House of Wisdom in Baghdad — al-Maʾmūn, the Graeco-Arabic translation movement, scholar cards (Banū Mūsā, Thābit ibn Qurra, al-Battānī, al-Karajī, Omar Khayyām).
6. "Greatest turning point" argument defending al-jabr/symbolic notation as the pivotal turning point, with five numbered premises and explicit counter-argument handling.
7. References — primary sources (al-Khwārizmī's Kitāb al-jabr, Brahmagupta, Fibonacci) and secondary scholarship (Berggren, Gandz, Lyons, Gutas, Boyer & Merzbach, Rashed, Katz, Høyrup, MacTutor).

Design: dark theme (#0A0A12 base, #0F1629 alternating, #F0E6D3 parchment text, #C9A84C gold accents), Playfair Display + Inter via Google Fonts, sticky nav with smooth-scroll anchors, IntersectionObserver scroll reveals, SVG gold dividers between sections, illuminated-manuscript radial-gradient background texture.
